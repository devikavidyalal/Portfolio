Cyberbullying Detection

Description:

Cyberbullying Detection is an AI-powered web app that analyzes text data to identify cyberbullying in online interactions. Using Logistic Regression, the model classifies harmful content and provides real-time predictions. The system is integrated with a Flask API for seamless deployment, enabling accurate and efficient cyberbullying detection.

Technologies Used

  • Python
  • Flask
  • Machine Learning
  • Logistic Regression
  • Natural Language Processing
