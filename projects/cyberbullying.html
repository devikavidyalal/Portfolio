Case File 02

Cyberbullying Detection

An NLP-powered machine learning application that detects cyberbullying from text in real time, helping identify harmful online interactions through intelligent text classification.

Project Overview

Cyberbullying Detection is a machine learning web application developed using Python and Flask. The system analyzes user-provided text and predicts whether it contains cyberbullying content using Natural Language Processing techniques and a Logistic Regression model.

The project focuses on transforming raw text into meaningful numerical features through TF-IDF vectorization, enabling the trained model to classify harmful content quickly and accurately through a simple web interface.

Key Features

  • Real-time cyberbullying prediction from user input.
  • Text preprocessing and feature extraction using TF-IDF.
  • Logistic Regression model for efficient text classification.
  • Flask-based web application for seamless interaction.
  • Model evaluated using accuracy, precision and recall metrics.

Technologies Used

Python Flask Machine Learning Natural Language Processing Logistic Regression TF-IDF Pandas
