Case File 03

SkillSense

An AI-powered career assistant that analyzes resumes, matches job descriptions, and provides personalized career guidance using semantic search, Retrieval-Augmented Generation (RAG), and Large Language Models.

Project Overview

SkillSense is an AI career coaching platform designed to help job seekers improve their career readiness. The application analyzes uploaded resumes, identifies strengths and missing skills, evaluates compatibility with job descriptions, and provides AI-driven career guidance.

The system uses a Flask backend following a service and repository architecture. Resume and job matching features are powered by Sentence Transformer embeddings, while the career assistant uses a Retrieval-Augmented Generation pipeline with FAISS vector search and Groq LLM.

Key Features

  • AI-based resume analysis with strengths, weaknesses, missing skills and improvement suggestions.
  • Semantic resume-to-job description matching using Sentence Transformers and cosine similarity.
  • RAG-powered career chatbot that retrieves relevant knowledge from FAISS vector database.
  • Secure authentication system with user sessions and JWT integration.
  • MongoDB-based storage for user data and analysis records.
  • Modular Flask architecture using routes, services and repositories.
