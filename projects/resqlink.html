ResQlink

A Flask-based emergency response platform that integrates machine learning, real-time alerts, travel safety monitoring and intelligent automation to provide rapid assistance during emergency situations.

Project Overview

ResQlink is a full-stack emergency response platform developed using Python, Flask and MongoDB to improve personal safety through intelligent automation and rapid emergency communication. The system enables users to instantly alert trusted contacts and access emergency assistance during critical situations.

The platform integrates voice-triggered emergency activation, location sharing, travel safety monitoring, real-time weapon detection using YOLO, automated image capture, SMS and email notifications, and nearby police station identification to provide a comprehensive emergency response solution.

Key Features

  • Voice-triggered emergency activation for hands-free assistance.
  • One-tap SOS with location sharing to trusted contacts.
  • Real-time weapon detection using the YOLO object detection model.
  • Automatic image capture during emergencies for evidence collection.
  • SMS and email notifications using Twilio integration.
  • Travel safety monitoring and nearby police station identification.
