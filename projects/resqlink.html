ResQlink

Description:

ResQlink is a smart safety web app that detects emergencies through a predefined codeword. When spoken, it triggers real-time alerts via email and SMS to emergency contacts, including live location, captured image, nearest police station details, and weapon detection.

If speaking isn't an option, a chatbot assists with distress signaling and travel monitoring. Leveraging AI and real-time communication, ResQlink ensures swift emergency response and enhanced personal security.

Technologies Used

  • Python
  • MongoDB
  • Flask
  • Machine Learning
  • Twilio API
